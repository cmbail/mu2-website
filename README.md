# MU-2 Flyers Association Website

Static GitHub Pages website for the MU-2 Flyers Association.

## Files

- `index.html` is the homepage.
- `about.html`, `membership.html`, `forum.html`, `events.html`, `resources.html`, `safety-training.html`, and `contact.html` are the main pages.
- `assets/css/styles.css` contains the shared design system.
- `assets/js/main.js` handles the mobile menu and static email-contact form.
- `assets/docs/fly-in-jet-air-2026.pdf` is the member fly-in flyer.
- `archive/current-site/` contains the full `wget` mirror of the previous website.

## GitHub Pages

Upload the root contents of this folder to a GitHub Pages repository. The site uses plain static HTML/CSS/JS and does not need a build step. The `.nojekyll` file is included so GitHub Pages serves the asset folders exactly as written.
